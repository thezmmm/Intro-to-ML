# Introduction to Machine Learning

This repository contains my work for the **Introduction to Machine Learning** course at **University of Helsinki**. It includes exercises, term project, and prerequisite knowledge examination materials. The repository is organized to reflect the course structure and my learning journey.

## Repository Structure

- `prerequisite-knowledge-exam/` – Solutions and notes for the prerequisite knowledge examination
- `exercise-set-1/` – Exercises and solutions for Exercise Set 1
- `exercise-set-2/` – Exercises and solutions for Exercise Set 2
- `exercise-set-3/` – Exercises and solutions for Exercise Set 3
- `iml25-term-project/` – Final term project for the course

## Topics Covered

- Supervised Learning (Linear Regression, Logistic Regression, Decision Trees, etc.)
- Unsupervised Learning (Clustering, PCA, etc.)
- Model Evaluation and Selection
- Practical Machine Learning Projects using Python (scikit-learn, pandas, matplotlib, etc.)
